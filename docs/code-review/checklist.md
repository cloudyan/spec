# checklist

在查看代码时，问自己以下基本问题：

- 我能**轻松理解**代码吗？
- 代码是否按照**编码标准/指南编写**？
- 相同的代码是否**重复**了两次以上？
- 我可以**轻松地对代码进行单元测试/调试**以找到根本原因吗？
- 这个函数或类是否**太大**？ 如果是，函数或类是否有太多职责？
- 如果您觉得上述任何问题的答案都不令人满意，那么您可以建议/建议代码更改。
