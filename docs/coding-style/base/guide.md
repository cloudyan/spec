### 提前设计

“磨刀不误砍柴功”，千万不要为了求快而直接写代码。

具体流程

- 阅读文档，分析需求
- 画原型图或草图（方便自己理解整体架构）
- 写大纲或伪代码（如果项目比较大还要细分模块）
- 实现细节
